HTML básico

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo.
Os elementos HTML ou também chamados de tags HTML , são utilizados para informar ao navegador que tipo de estrutura que está sendo construída.

Tag html

Essa tag está sempre no começo e no fim da página web.
Ela limita onde o código começa e termina.

Tag head

Utilizada para definir o começo e o fim do cabeçalho de um documento HTML.

Tag meta

Essa tag traz dados básicos sobre o documento, como autor e data de criação.

Tag title

Define o título do documento HTML .
Esse é o nome que é exibido no topo da janela do navegador quando a página é aberta.

Tag body

Indica para o navegador o corpo do documento HTML, ou seja, a parte principal da página web.

Tag para criar títulos

As tags HTML que usamos para criar títulos são: h1, h2, h3, h4, h5 e h6 . O nível de importância é diferente para cada uma, onde h1 tem o nível mais alto e h6 o mais baixo

Tag p

Define o começo e o fim de um parágrafo.

Tag strong

Forma o texto com negrito.

Tag br

Define uma quebra de linha.

Tag em

Da ênfase a um texto em itálico
